== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Events > Settings to configure the plugin
4. Start creating events!

== Frequently Asked Questions ==

= What are the requirements? =

WordPress 5.0 or higher and PHP 7.2 or higher. No external dependencies required.

= How do I display the calendar? =

Use the shortcode `[twec_calendar]` or visit the events archive page. You can specify the view: `[twec_calendar view="month"]`

= Can I hide past events? =

Yes! Go to Events > Settings and enable "Hide Past Events".
