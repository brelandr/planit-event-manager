# Features Comparison: The WordPress Event Calendar vs The Events Calendar (Free)

## ✅ All Free Features Included

### Core Calendar Views
- ✅ **Month View** - Traditional calendar grid display
- ✅ **List View** - Chronological list of events
- ✅ **Day View** - All events for a specific day
- ✅ **Week View** - Weekly view with hourly breakdown *(Bonus - not in free version)*
- ✅ **Year View** - Annual overview *(Bonus - not in free version)*

- ✅ **Event Creation & Editing** - Full CRUD operations

- ✅ **Venue Management** - Separate post type for venues

- ✅ **Organizer Contact** - Phone, email, website

### Organization & Filtering
- ✅ **Event Categories** - Hierarchical taxonomy
- ✅ **Event Tags** - Non-hierarchical taxonomy

- ✅ **Event Search** - Search events by keyword
- ✅ **WordPress Search Integration** - Events appear in site search

### Display Options
- ✅ **Hide Past Events** - Option to hide expired events
- ✅ **Events Per Page** - Configurable pagination
- ✅ **Responsive Design** - Mobile-friendly layouts
- ✅ **Custom Templates** - Overridable template system

### Widgets
- ✅ **Upcoming Events Widget** - Display upcoming events in sidebar
- ✅ **Featured Events Widget** - Display featured events *(Pro feature)*

- ✅ **Calendar Shortcode** - `[twec_calendar]` with view options
- ✅ **List Shortcode** - `[twec_list]` with filtering options
